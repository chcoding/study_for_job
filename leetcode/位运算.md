# 位运算

#### 返回n的最后一位1

`lowbit(n) = n & -n`



#### 判断一个数是否为2的幂次方

- *n* 二进制最高位为 1，其余所有位为 0；
- n−1 二进制最高位为 0，其余所有位为 1；

```c++
n > 0 and n & (n - 1) == 0
```



#### 交换两个数

1. 加减运算
2. 异或



##### 加减运算

```c++
a = a+b;
b = a -b;
a = a -b;
```



##### 异或运算

```c++
a = a ^ b
b = a ^ b = a ^ b ^ b = a
a = a ^ b = a ^ b ^ a = b
```
