# 单调栈

用 O(n) 复杂度的一重遍历找到**每个元素**前后最近的**更小/大**元素位置



#### 代码框架

```c++
stack<int> st;
for(int i = 0; i < nums.size(); i++)
{
	while(!st.empty() && st.top() > nums[i])
	{
		st.pop();
	}
	st.push(nums[i]);
}
```
